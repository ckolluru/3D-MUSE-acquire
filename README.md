3D-MUSE-acquire
==========

Running the software
------------

Install all necessary pacakages

```
pycromanager
pyfirmata
pyqt5
pyqt5-tools
glob
itk-montage
zarr
numpy
tqdm
opencv-python
Pillow
matplotlib
scikit-image
```

- Go to ```bin\``` folder and run the batch script
- It will start a command window, do not close that at any time when using the application
